# 考试结果系统

这是一个使用 React 构建的考试结果展示系统，模仿了考试阅卷和结果展示页面。

## 功能特点

- **阅卷页面**：显示考试结束后的阅卷等待页面，带有动态加载效果
- **结果页面**：展示考试结果，包括是否通过、考试用时等详细信息
- **独立页面**：两个页面通过不同的URL独立访问，不会自动跳转
- **响应式设计**：适配不同屏幕尺寸

## 安装

```bash
npm install
```

## 运行

```bash
npm start
```

应用将在 [http://localhost:3000](http://localhost:3000) 打开。

## 页面说明

### 1. 阅卷页面 (`/grading`)
- 显示"考试结束，阅卷中"的提示
- 包含动态加载动画
- 独立页面，不会自动跳转

### 2. 结果页面 (`/result`)
- 显示考试结果（未通过）
- 展示考生信息
- 显示考试用时和题目数量
- 提供"考试报告"按钮
- 包含水印效果

## 访问页面

启动应用后，可以通过以下URL访问不同页面：

### 主要页面（手写还原版本，使用用户插图）
- **阅卷页面**：`http://localhost:3000/grading` ⭐
- **结果页面**：`http://localhost:3000/result` ⭐

### 备用页面
- 阅卷页面（备用）：`http://localhost:3000/grading-new`
- 结果页面（备用）：`http://localhost:3000/result-new`

> **说明**：主要页面使用手写代码完全还原了原设计，结果页面的插图使用了用户提供的定制图片（女生在电脑前工作）。

## 技术栈

- React 18
- React Router DOM 6
- CSS3 动画
- SVG 插图
